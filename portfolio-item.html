OnlineBusiness Card

project

Creating a visually appealing and interactive business card using HTML and CSS was a rewarding project that showcases my web development skills. The business card design features a clean and modern aesthetic, with a focus on user experience. One of the standout features is the interactive image, which dynamically responds to user interactions.

On hover, the image smoothly shifts from left to right, adding a playful yet professional touch. Additionally, the image transitions from a light pink hue to a rich dark pink, creating a visually striking effect that captures attention.

This project not only demonstrates my proficiency in HTML and CSS but also highlights my ability to implement creative solutions to enhance user engagement and interaction.
